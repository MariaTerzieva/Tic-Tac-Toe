    Tic-Tac-Toe

    Simple Tic Tac Toe game that can be played in the browser

    How to use the web application:

    • Clone this repository by typing git clone github.com/MariaTerzieva/Tic-Tac-Toe.git in your terminal

    • Run cd Tic-Tac-Toe

    • Run bundle install

    • Run the tests with rake test

    • Run rails s

    • In your browser (preferably Google Chrome) go to localhost:3000

    • Enjoy!

    Rails version: 4.2.1
